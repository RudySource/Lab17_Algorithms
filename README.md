# Что такое алгоритм

**Алгоритм** - это точная последовательность действий, направленная на решение определённой задачи.

Алгоритмы используются практически во всех программах и системах.

## Основные свойства алгоритма

1. **Дискретность** - алгоритм состоит из отдельных шагов
2. **Детерминированность** - каждый шаг однозначно определён
3. **Конечность** - алгоритм выполняется за конечное число шагов
4. **Массовость** - применим к целому классу задач
5. **Результативность** - выполнение приводит к результату

> [!TIP]
> Почти любую задачу можно сначала описать как алгоритм, а затем реализовать её в коде.

# Блок-схемы

Блок-схема - это графическое представление алгоритма.

Она показывает последовательность действий программы с помощью специальных элементов.

## Основные элементы блок-схем

| Элемент        | Назначение                 |
| -------------- | -------------------------- |
| Овал           | Начало или конец алгоритма |
| Прямоугольник  | Выполнение действия        |
| Ромб           | Условие                    |
| Параллелограмм | Ввод или вывод данных      |
| Стрелки        | Направление выполнения     |

> [!NOTE]
> В ромбе всегда формулируется вопрос, на который можно ответить **Да** или **Нет**.
